# Dependency Check Guide

Ensure you have all the things and proper versions installed for React Native development.


### Node
Run:
```
node -v
```

Result:
```
v8.11.2
```

### react-native-cli
Run:
```
react-native -v
```

Result:
```
react-native-cli: 2.0.1
react-native: n/a - not inside a React Native project directory
```
